# Final-Projct-B
Analyzing the Security of Healthcare Systems Using Artificial Intelligence
Introduction
In this project I made a machine learning and deep learning system to find the severity levels of cybersecurity problems in healthcare systems I chose this topic because hospitals and healthcare companies store private patient information and digital medical records so cybersecurity is very important The main goal of this project was to build smart models that can automatically predict how dangerous a cybersecurity threat is by using text descriptions numbers and categories This project uses Machine Learning ML Deep Learning DL and Natural Language Processing NLP to look at cybersecurity problems and choose their severity levels automatically

Dataset Description
The dataset in this project has 1515 rows and 11 columns about healthcare cybersecurity problems from 2011 to 2026 The most important features in the dataset are Description which is the text explaining the cybersecurity problem CVSS Score which is the number for the severity score Attack Vector which is the attack method Weakness which is the type of security problem Severity which is the final label we want to predict The dataset also has information and references for the problems to help explain the type and danger of each threat Before training the models I cleaned and prepared the data by changing text to lowercase letters removing extra symbols and fixing missing data I changed missing categories to Unknown and filled missing numbers with the median value To use the text data I used TF IDF Vectorization to change the descriptions into numbers for the machine learning models Finally I divided the dataset into 80 percent for training and 20 percent for testing

Models Used
In this project I trained and compared three machine learning models Logistic Regression Random Forest Classifier K Nearest Neighbors KNN I also built a Deep Learning Neural Network using TensorFlow and Keras The data preparation pipeline combined TF IDF Vectorization for text features One Hot Encoding for categories and numerical CVSS scores I tested the models using Accuracy Precision Recall and F1 Score

Neural Network Architecture
The neural network design had a Dense layer with 128 neurons a Dropout layer to stop overfitting a Dense layer with 64 neurons another Dropout layer and a Softmax output layer I trained the model for 10 epochs with a batch size of 32 Before training I changed the labels into numbers using Label Encoding and changed the features into numerical arrays for the deep learning models During training I checked the training accuracy validation accuracy training loss and validation loss to see how the model works and to stop overfitting The Neural Network learned difficult patterns but Logistic Regression was still better and faster

Final Results and Comparison
Here are the final results for the models Logistic Regression attained 97.67 percent Accuracy and 97.64 percent F1 Score KNN attained 95.00 percent Accuracy and 95.02 percent F1 Score Random Forest attained 85.00 percent Accuracy and 84.72 percent F1 Score Neural Network attained 83.33 percent Accuracy and 82.54 percent F1 Score The results showed that Logistic Regression was the best model because it had the highest accuracy and F1 score The final comparison showed big differences between the models Logistic Regression gave the best results because it worked very well with TF IDF text features and it was very fast KNN was also good but it was a bit less accurate because text data can make distance algorithms less effective Random Forest worked well with different data types but its accuracy was lower than Logistic Regression and KNN The Neural Network needed more time and computer power but it was not better than Logistic Regression on this dataset An important thing I noticed is that cleaning the data and preparing the features changed the results a lot Cleaning text descriptions and using TF IDF features made the classification accuracy much better

What I Learned
From this project I learned how we can use artificial intelligence for real world healthcare cybersecurity problems I got good experience in data cleaning NLP preparation TF IDF Vectorization machine learning pipelines model evaluation and deep learning with TensorFlow and Keras I also learned that complex models are not always the best choice In this project Logistic Regression was better than the Neural Network because the dataset and the features were perfect for traditional machine learning methods This project shows how AI can help healthcare cybersecurity by choosing threat levels automatically and making security analysis faster
